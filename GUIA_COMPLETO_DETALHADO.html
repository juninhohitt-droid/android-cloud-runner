🛠️ Android Studio - Guia Completo

1
Baixar Android Studio

Passo 1.1: Abrir o site oficial

  • Abra seu navegador (Chrome, Firefox, Edge, etc.)
  • Digite na barra de endereço: https://developer.android.com/studio
  • Pressione Enter

Passo 1.2: Clicar em Download

  • Você verá um botão grande azul: "Download Android Studio"
  • Clique nele

Passo 1.3: Aceitar termos

  • Aparecerá uma página com termos de serviço
  • Marque a caixa: "I have read and agree..."
  • Clique em "Download Android Studio"
⏱️ Tempo: 5-15 minutos (depende da velocidade de internet)
Tamanho: ~2 GB
2
Instalar Android Studio

Passo 2.1: Localizar o arquivo

  • Abra a pasta Downloads
  • Procure por: android-studio-XXXX.exe

Passo 2.2: Executar instalador

  • Duplo clique no arquivo
  • Se aparecer pergunta: "Do you want to allow...", clique "Yes"

Passo 2.3: Seguir o assistente

  • Clique "Next" na primeira tela
  • Selecione "Standard" em "Install Type"
  • Clique "Next"
  • Deixe as pastas padrão e clique "Next"
  • Clique "Install"
⚠️ IMPORTANTE: Não feche a janela durante a instalação!
Isso pode levar 10-20 minutos.

Passo 2.4: Finalizar

  • Quando terminar, clique "Finish"
⏱️ Tempo: 15-30 minutos
3
Primeira Execução

Passo 3.1: Android Studio vai abrir

  • Após clicar em "Finish", Android Studio abrirá
  • Isso pode levar alguns minutos

Passo 3.2: Tela de Boas-vindas

  • Você verá a tela inicial com opções
  • Clique em "New Project" (ou qualquer opção)

Passo 3.3: Baixar componentes

  • Android Studio vai baixar componentes necessários
  • Você verá mensagens como "Downloading SDK..."
  • Aguarde até terminar (5-10 minutos)
✓ Quando terminar, você verá a tela principal do Android Studio
⏱️ Tempo: 10-20 minutos
4
Criar Emulador Android

Passo 4.1: Abrir Device Manager

  • No Android Studio, clique no menu "Tools"
  • Procure por "Device Manager"
  • Clique nele

Passo 4.2: Criar novo dispositivo

  • Você verá a janela "Device Manager"
  • Clique em "Create device" (ou "Create Virtual Device")

Passo 4.3: Escolher modelo

  • Aparecerá uma lista de modelos de telefones
  • Selecione "Pixel 5" (recomendado)
  • Clique em "Next"

Passo 4.4: Escolher versão do Android

  • Aparecerá uma lista de versões do Android
  • Selecione "Android 13" (ou mais recente)
  • Se aparecer botão "Download", clique para baixar
  • Clique em "Next"

Passo 4.5: Verificar configurações

  • Você verá um resumo das configurações
  • Deixe tudo como está
  • Clique em "Finish"
✓ Emulador criado com sucesso!
⏱️ Tempo: 10-30 minutos (depende do download)
5
Iniciar o Emulador

Passo 5.1: Abrir Device Manager

  • No Android Studio, vá para Tools → Device Manager

Passo 5.2: Iniciar emulador

  • Você verá o emulador na lista (ex: "Pixel 5")
  • Clique no botão de "Play" (triângulo verde)

Passo 5.3: Aguardar inicialização

  • Uma janela vai abrir mostrando o emulador iniciando
  • Você verá a tela preta do Android iniciando
  • Aguarde 2-5 minutos
✓ Quando terminar, você verá a tela inicial do Android com relógio, data e ícones de apps
⏱️ Tempo: 2-5 minutos
6
Configurações Importantes

Aumentar Performance (Opcional)

  • Se o emulador estiver lento, você pode aumentar a RAM
  • No Device Manager, clique em "Edit" (lápis)
  • Procure por "RAM" e aumente para 4 GB
  • Clique em "Finish"

Ativar Modo Desenvolvedor

  • Dentro do emulador, abra Configurações
  • Procure por "Sobre o telefone"
  • Clique 7 vezes em "Número da compilação"
  • Mensagem: "Você agora é um desenvolvedor!"
ℹ️ Isso permite instalar APKs de fontes desconhecidas

🚀 Android Cloud Runner - Guia Completo

1
Preparar o Emulador

Passo 1.1: Verificar se emulador está rodando

  • Abra Android Studio
  • Vá para Tools → Device Manager
  • Procure pelo seu emulador (ex: "Pixel 5")
  • Se houver ícone "Play", clique para iniciar
  • Aguarde 2-5 minutos
⚠️ IMPORTANTE: Deixe o emulador aberto enquanto usa Android Cloud Runner!
Não feche a janela do Android Studio.
2
Executar Android Cloud Runner

Passo 2.1: Abrir a pasta do projeto

  • Localize a pasta android-cloud-runner-portable
  • Abra a pasta

Passo 2.2: Abrir PowerShell

  • Clique direito em um espaço vazio da pasta
  • Selecione "Open PowerShell here"
  • Uma janela preta vai abrir

Passo 2.3: Executar o script

  • Na janela do PowerShell, digite:
START_PYTHON39.bat
  • Pressione Enter

Passo 2.4: Aguardar inicialização

  • Você verá mensagens como:
[OK] Python encontrado [OK] pip encontrado [*] Instalando dependencias... [OK] Dependencias instaladas [*] Iniciando Android Cloud Runner...
✓ Quando ver: "📱 Acesse: http://localhost:5000"
Significa que está pronto!
⚠️ Não feche esta janela! Ela precisa ficar aberta.
⏱️ Tempo: 2-5 minutos
3
Acessar a Interface Web

Passo 3.1: Abrir o navegador

  • Abra Chrome, Firefox, Edge ou Safari

Passo 3.2: Digitar URL

  • Na barra de endereço, digite:
http://localhost:5000
  • Pressione Enter
✓ A página vai carregar com a interface do Android Cloud Runner
⏱️ Tempo: 5 segundos
4
Fazer Upload de um APK

Passo 4.1: Preparar o arquivo APK

  • Você precisa ter um arquivo .apk no seu computador
  • Pode ser um app que você desenvolveu ou um app da internet

Passo 4.2: Método 1 - Arrastar e Soltar

  • Localize o arquivo .apk no seu computador
  • Na interface, procure pela área que diz: "Arrastar APK aqui"
  • Arraste o arquivo e solte nessa área
  • O upload vai começar

Passo 4.3: Método 2 - Clique para Selecionar

  • Clique na área de upload
  • Uma janela vai abrir
  • Navegue até seu arquivo .apk
  • Clique em "Abrir"

Passo 4.4: Acompanhar upload

  • Você verá uma barra de progresso
  • Mensagens como "Uploading..."
  • Quando terminar: "Upload concluído!"
⏱️ Tempo: 30 segundos a 2 minutos (depende do tamanho)
5
Instalar o APK

Passo 5.1: Após upload

  • O arquivo aparecerá na lista
  • Você verá um botão "Instalar"

Passo 5.2: Clicar em Instalar

  • Clique no botão "Instalar"
  • Você verá: "Instalando... Por favor aguarde"

Passo 5.3: Acompanhar instalação

  • Apps pequenos: 10-30 segundos
  • Apps médios: 1-3 minutos
  • Apps grandes: 5+ minutos
✓ Quando terminar: "APK instalado com sucesso!"

Passo 5.4: Verificar no emulador

  • Olhe para a janela do Android Studio
  • Você pode ver a tela de instalação
  • Quando terminar, o app está pronto!
6
Abrir o App

Passo 6.1: Usar o botão "Abrir"

  • Na interface, procure pelo app na lista
  • Clique no botão "Abrir"
  • O app vai abrir no emulador automaticamente

Passo 6.2: Abrir manualmente

  • Se o botão não funcionar, abra manualmente:
  • Olhe para a janela do emulador
  • Procure pela Gaveta de Apps (ícone de grade)
  • Clique nela
  • Procure pelo seu app
  • Clique para abrir
✓ Seu app está rodando no emulador!
Agora você pode testar todas as funcionalidades.
7
Acessar Android via VNC

O que é VNC?

VNC permite você ver e controlar o Android através do navegador, sem precisar da janela do Android Studio.

Passo 7.1: Abrir VNC

  • Na interface do Android Cloud Runner, procure por:
  • Um botão "Abrir VNC" ou "View Android"
  • Ou um link: "http://localhost:6080"
  • Clique nele

Passo 7.2: Você verá o Android no navegador

  • Uma nova aba vai abrir
  • Você verá a tela do Android
  • Pode clicar, digitar, arrastar, etc.
ℹ️ Isso é útil para ver o Android sem a janela do Android Studio

☁️ Deploy na Nuvem (Railway)

1
Criar Conta GitHub

Passo 1.1: Abrir GitHub

  • Abra seu navegador
  • Digite: https://github.com/signup
  • Pressione Enter

Passo 1.2: Preencher dados

  • Email: seu email
  • Password: sua senha
  • Username: seu nome de usuário

Passo 1.3: Confirmar email

  • GitHub vai enviar um email de confirmação
  • Abra seu email e clique no link
⏱️ Tempo: 5 minutos
2
Criar Repositório

Passo 2.1: Fazer login

  • Faça login em https://github.com

Passo 2.2: Novo repositório

  • Clique no ícone "+" no canto superior direito
  • Selecione "New repository"

Passo 2.3: Preencher dados

  • Repository name: android-cloud-runner
  • Description: Android Cloud Runner - Portátil
  • Selecione "Public"

Passo 2.4: Criar

  • Clique em "Create repository"
⏱️ Tempo: 3 minutos
3
Fazer Upload para GitHub

Passo 3.1: Abrir PowerShell

  • Na pasta android-cloud-runner-portable
  • Clique direito e selecione "Open PowerShell here"

Passo 3.2: Executar comandos

  • Digite os comandos abaixo (um por um):
git init git add . git commit -m "Initial commit" git remote add origin https://github.com/SEU_USERNAME/android-cloud-runner.git git branch -M main git push -u origin main

Substituir SEU_USERNAME pelo seu nome de usuário do GitHub

⏱️ Tempo: 5 minutos
4
Criar Conta Railway

Passo 4.1: Abrir Railway

  • Abra seu navegador
  • Digite: https://railway.app
  • Pressione Enter

Passo 4.2: Fazer login com GitHub

  • Clique em "Login"
  • Selecione "Login with GitHub"
  • Autorize Railway a acessar sua conta
⏱️ Tempo: 3 minutos
5
Fazer Deploy

Passo 5.1: Novo projeto

  • No painel do Railway, clique em "New Project"

Passo 5.2: Deploy do GitHub

  • Selecione "Deploy from GitHub repo"
  • Procure por android-cloud-runner
  • Clique para selecionar

Passo 5.3: Aguardar deploy

  • Railway vai começar o deploy automaticamente
  • Você verá mensagens de progresso
  • Aguarde 3-5 minutos
✓ Quando terminar, você verá "Deployment successful"
⏱️ Tempo: 5-10 minutos
6
Obter URL Pública

Passo 6.1: Acessar Settings

  • No painel do Railway, vá para seu projeto
  • Clique em "Settings" (ícone de engrenagem)

Passo 6.2: Procurar Domains

  • Procure por "Domains"
  • Você verá uma URL como:
android-cloud-runner-production.up.railway.app

Passo 6.3: Copiar URL

  • Copie essa URL
  • Abra em seu navegador
✓ Seu Android Cloud Runner está rodando na nuvem!
Agora você pode acessar de qualquer lugar!
7
