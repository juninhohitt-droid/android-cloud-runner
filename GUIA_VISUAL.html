O que é Android Cloud Runner?

Android Cloud Runner é uma solução portátil que permite rodar seu APK em um Android real, sem Docker, sem limitações.

Você pode usar localmente no seu computador ou fazer deploy na nuvem (Railway) para acessar de qualquer lugar.

✨
Características

✅ Sem Docker — Funciona em Windows 10 antigo

✅ Portátil — Leve e fácil de usar

✅ Android Real — Não é simulação

✅ Interface Web — Acesse via navegador

✅ Acesso Remoto — Railway gratuito

✅ 100% Gratuito — Open-source

⏱️
Tempo Estimado

Primeira vez: 30-45 minutos

Próximas vezes: 5 minutos (apenas executar START.bat)

🚀
Próximos Passos

Selecione seu sistema operacional acima (Windows, Mac ou Linux) e siga o guia passo a passo!

🪟 Guia para Windows

1
Instalar Python

1. Abra https://www.python.org/downloads/

2. Clique no botão amarelo "Download Python 3.x.x"

3. Execute o instalador

⚠️ IMPORTANTE: Marque a caixa "Add Python to PATH" durante a instalação!

4. Clique em "Install Now"

5. Reinicie o computador

2
Instalar Android Studio

1. Abra https://developer.android.com/studio

2. Clique em "Download Android Studio"

3. Execute o instalador

4. Siga as instruções de instalação

3
Criar Emulador Android

1. Abra Android Studio

2. Vá para Tools → Device Manager

3. Clique em "Create device"

4. Selecione um modelo (ex: Pixel 5)

5. Clique em "Next"

6. Selecione uma versão do Android (ex: Android 13)

7. Clique em "Finish"

4
Descompactar o ZIP

1. Localize o arquivo android-cloud-runner-portable.zip

2. Clique direito e selecione "Extract All..."

3. Escolha um local (ex: Desktop ou Documents)

4. Clique em "Extract"

5
Abrir Terminal na Pasta

1. Abra a pasta android-cloud-runner-portable

2. Clique direito em um espaço vazio

3. Selecione "Open PowerShell here"

6
Executar START.bat

Na janela do PowerShell, digite:

START.bat

Pressione Enter

Aguarde as mensagens:

✓ Python encontrado
✓ pip encontrado
📦 Instalando dependências...
✓ Dependências instaladas
🚀 Iniciando Android Cloud Runner...
📱 Acesse: http://localhost:5000
7
Abrir no Navegador

1. Abra seu navegador (Chrome, Firefox, Edge, Safari)

2. Na barra de endereço, digite:

http://localhost:5000

3. Pressione Enter

4. Você verá a interface do Android Cloud Runner!

🍎 Guia para Mac

1
Instalar Homebrew

Abra Terminal e digite:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2
Instalar Python

No Terminal, digite:

brew install python3
3
Instalar Android Studio

1. Abra https://developer.android.com/studio

2. Clique em "Download Android Studio"

3. Ou use Homebrew:

brew install android-studio
4
Criar Emulador Android

2. Vá para Tools → Device Manager

3. Clique em "Create device"

4. Siga as instruções

5
Descompactar e Executar

cd ~/Downloads/android-cloud-runner-portable

chmod +x start.sh ./start.sh

4. Aguarde a inicialização

6
Abrir no Navegador

Abra seu navegador e digite:

http://localhost:5000

☁️ Deploy no Railway (Acesso Remoto Gratuito)

1
Criar Conta GitHub

1. Abra https://github.com/signup

2. Preencha os dados

3. Confirme o email

2
Criar Repositório

1. Faça login no GitHub

2. Clique no ícone "+" → "New repository"

3. Nome: android-cloud-runner

4. Marque "Public"

5. Clique em "Create repository"

3
Fazer Upload para GitHub

Abra Terminal na pasta e digite:

git init git add . git commit -m "Initial commit" git remote add origin https://github.com/SeuUsuario/android-cloud-runner.git git branch -M main git push -u origin main
4
Criar Conta Railway

1. Abra https://railway.app

2. Clique em "Login"

3. Selecione "Login with GitHub"

4. Autorize Railway

5
Fazer Deploy

1. No painel do Railway, clique em "New Project"

2. Selecione "Deploy from GitHub repo"

3. Selecione seu repositório

4. Clique em "Deploy"

5. Aguarde 3-5 minutos

6
Obter URL Pública

1. No painel do Railway, vá para "Settings"

2. Procure por "Domains"

3. Você verá uma URL como:

android-cloud-runner-production.up.railway.app

4. Abra no navegador para acessar remotamente!

💡 Dica: Railway oferece 500 horas/mês grátis. Suficiente para usar 16+ horas por dia!
