🛠️ Android Studio - Guia Completo

🪟 Windows
1
Baixar Android Studio (Windows)

Passo 1.1: Abrir o site oficial

  • Abra seu navegador
  • Digite: https://developer.android.com/studio
  • Pressione Enter

Passo 1.2: Clicar em Download

  • Clique no botão azul: "Download Android Studio"

Passo 1.3: Aceitar termos

  • Marque a caixa: "I have read and agree..."
  • Clique em "Download Android Studio"
⏱️ Tempo: 5-15 minutos (2 GB)
2
Instalar Android Studio (Windows)

Passo 2.1: Executar instalador

  • Abra a pasta Downloads
  • Duplo clique em android-studio-XXXX.exe
  • Se aparecer pergunta, clique "Yes"

Passo 2.2: Seguir o assistente

  • Clique "Next"
  • Selecione "Standard"
  • Clique "Next" novamente
  • Deixe as pastas padrão
  • Clique "Install"
⏱️ Tempo: 15-30 minutos
3
Primeira Execução (Windows)

Passo 3.1: Android Studio vai abrir

  • Clique em "Finish"
  • Aguarde alguns minutos

Passo 3.2: Baixar componentes

  • Android Studio vai baixar componentes
  • Você verá "Downloading SDK..."
  • Aguarde 5-10 minutos
⏱️ Tempo: 10-20 minutos
4
Criar Emulador (Windows)

Passo 4.1: Abrir Device Manager

  • Clique em "Tools"
  • Procure por "Device Manager"
  • Clique nele

Passo 4.2: Criar novo dispositivo

  • Clique em "Create device"
  • Selecione "Pixel 5"
  • Clique "Next"

Passo 4.3: Escolher versão Android

  • Selecione "Android 13"
  • Se aparecer "Download", clique para baixar
  • Clique "Next"

Passo 4.4: Finalizar

  • Clique "Finish"
⏱️ Tempo: 10-30 minutos
5
Iniciar Emulador (Windows)

Passo 5.1: Device Manager

  • Vá para Tools → Device Manager

Passo 5.2: Clicar em Play

  • Clique no botão de "Play" verde
  • Aguarde 2-5 minutos
✓ Emulador pronto!
⏱️ Tempo: 2-5 minutos
1
Baixar Android Studio (Mac)

Passo 1.1: Abrir o site

  • Abra Safari ou Chrome
  • Digite: https://developer.android.com/studio

Passo 1.2: Download para Mac

  • Clique em "Download Android Studio"
  • Selecione a versão para Mac (Intel ou Apple Silicon)
  • Se tem Mac M1/M2/M3, escolha "Apple Silicon"
  • Senão, escolha "Intel"

Passo 1.3: Aceitar termos

  • Marque a caixa de concordância
  • Clique em Download
⏱️ Tempo: 5-15 minutos (2 GB)
2
Instalar Android Studio (Mac)

Passo 2.1: Abrir o arquivo DMG

  • Abra a pasta Downloads
  • Duplo clique em android-studio-XXXX.dmg

Passo 2.2: Arrastar para Applications

  • Uma janela vai abrir
  • Você verá o ícone do Android Studio
  • Arraste para a pasta "Applications"
  • Aguarde a cópia (pode levar alguns minutos)

Passo 2.3: Ejetar

  • Clique no ícone de ejetar
⏱️ Tempo: 10-20 minutos
3
Primeira Execução (Mac)

Passo 3.1: Abrir Android Studio

  • Abra Finder
  • Vá para Applications
  • Duplo clique em "Android Studio"

Passo 3.2: Autorizar

  • Se aparecer pergunta, clique "Open"

Passo 3.3: Baixar componentes

  • Android Studio vai baixar componentes
  • Aguarde 10-15 minutos
⏱️ Tempo: 15-25 minutos
4
Criar Emulador (Mac)

Passo 4.1: Device Manager

  • Clique em "Tools" no menu
  • Selecione "Device Manager"

Passo 4.2: Criar dispositivo

  • Clique em "Create device"
  • Selecione "Pixel 5"
  • Clique "Next"

Passo 4.3: Versão Android

  • Selecione "Android 13"
  • Clique "Next"

Passo 4.4: Finalizar

  • Clique "Finish"
⏱️ Tempo: 15-30 minutos
5
Iniciar Emulador (Mac)

Passo 5.1: Device Manager

  • Vá para Tools → Device Manager

Passo 5.2: Play

  • Clique no botão "Play"
  • Aguarde 3-5 minutos
✓ Emulador pronto no Mac!

Instalar Android Studio (Linux)

  • Pressione Ctrl + Alt + T
  • Ou abra o Terminal manualmente

Passo 2.2: Navegar para Downloads

cd ~/Downloads

Passo 2.3: Extrair arquivo

tar -xzf android-studio-XXXX-linux.tar.gz

Passo 2.4: Mover para /opt

sudo mv android-studio /opt/

Passo 2.5: Executar

/opt/android-studio/bin/studio.sh

🚀 Android Cloud Runner - Guia Completo

As instruções abaixo funcionam igualmente em Windows, Mac e Linux

1
Preparar o Emulador

Passo 1.1: Verificar emulador

  • Abra Android Studio
  • Vá para Tools → Device Manager
  • Clique em "Play" para iniciar
  • Aguarde 2-5 minutos
⚠️ Deixe o emulador aberto enquanto usa Android Cloud Runner!
2
Executar Android Cloud Runner

Passo 2.1: Abrir Terminal

  • Windows: Clique direito na pasta → "Open PowerShell here"
  • Mac: Clique direito na pasta → "New Terminal at Folder"
  • Linux: Clique direito na pasta → "Open in Terminal"

Passo 2.2: Executar script

START_PYTHON39.bat

Passo 2.3: Aguardar

  • Você verá mensagens de inicialização
  • Quando ver "📱 Acesse: http://localhost:5000", está pronto!
⏱️ Tempo: 2-5 minutos
3
Acessar Interface Web

Passo 3.1: Abrir navegador

  • Abra Chrome, Firefox, Safari ou Edge

Passo 3.2: Digitar URL

http://localhost:5000

Passo 3.3: Pressionar Enter

  • A interface vai carregar
⏱️ Tempo: 5 segundos
4
Fazer Upload de APK

Passo 4.1: Arrastar e soltar

  • Localize seu arquivo .apk
  • Arraste para a área de upload
  • Solte o arquivo

Passo 4.2: Acompanhar upload

  • Você verá uma barra de progresso
  • Aguarde até terminar
⏱️ Tempo: 30 segundos a 2 minutos
5
Instalar APK

Passo 5.1: Clicar em Instalar

  • Clique no botão "Instalar"
  • Você verá "Instalando..."

Passo 5.2: Aguardar

  • Apps pequenos: 10-30 segundos
  • Apps médios: 1-3 minutos
  • Apps grandes: 5+ minutos
✓ Quando ver "APK instalado com sucesso!", está pronto!
6
Abrir e Testar App

Passo 6.1: Clique em Abrir

  • Clique no botão "Abrir"
  • O app vai abrir no emulador

Passo 6.2: Testar

  • Teste todas as funcionalidades
  • Clique em botões
  • Preencha formulários
✓ Seu app está rodando!

☁️ Deploy na Nuvem (Railway)

As instruções abaixo funcionam igualmente em Windows, Mac e Linux

1
Criar Conta GitHub

Passo 1.1: Abrir GitHub

  • Abra seu navegador
  • Digite: https://github.com/signup

Passo 1.2: Preencher dados

  • Email, senha e nome de usuário

Passo 1.3: Confirmar email

  • GitHub vai enviar email
  • Clique no link de confirmação
⏱️ Tempo: 5 minutos
2
Criar Repositório

Passo 2.1: Fazer login

  • Faça login em https://github.com

Passo 2.2: Novo repositório

  • Clique no ícone "+" no canto superior direito
  • Selecione "New repository"

Passo 2.3: Preencher dados

  • Repository name: android-cloud-runner
  • Selecione "Public"

Passo 2.4: Criar

  • Clique em "Create repository"
⏱️ Tempo: 3 minutos
3
Fazer Deploy no Railway

Passo 3.1: Abrir Railway

  • Abra seu navegador
  • Digite: https://railway.app

Passo 3.2: Login com GitHub

  • Clique em "Login"
  • Selecione "Login with GitHub"
  • Autorize Railway

Passo 3.3: Deploy

  • Clique em "New Project"
  • Selecione "Deploy from GitHub repo"
  • Procure por "android-cloud-runner"
  • Clique para selecionar

Passo 3.4: Aguardar

  • Railway vai fazer o deploy automaticamente
  • Aguarde 3-5 minutos
✓ Quando ver "Deployment successful", está pronto!
⏱️ Tempo: 10-15 minutos
4
Obter URL Pública

Passo 4.1: Settings

  • No painel do Railway, clique em "Settings"

Passo 4.2: Procurar Domains

  • Procure por "Domains"
  • Você verá uma URL como:
android-cloud-runner-production.up.railway.app

Passo 4.3: Usar

  • Copie a URL
  • Abra em seu navegador
✓ Seu Android Cloud Runner está na nuvem!
Acesse de qualquer lugar do mundo!
